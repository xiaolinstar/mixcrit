# P0.9 写实调酒页与原料权重 Spec

> 创建日期：2026-06-16  
> 产品阶段：P0.9  
> 目标发布标识：0.0.10 (10+)  
> 前置基线：P0.8 本地提交 `126c94b`

---

## 1. 背景

P0.8 已将调酒主舞台迁移到 SpriteKit，并接入较真实的首页与部分调酒资产。验收后仍有两个明显问题：

- 首页视觉提升明显，但调酒页本体仍偏“UI 面板”，真实台面、杯具、入料反馈不足。
- 底部原料图像偏小，动作按钮视觉权重过高，不符合“原料是主要操作物件”的玩法定位。

P0.9 聚焦继续打磨 Mojito 单杯调酒体验，不扩展新酒单，不引入经营系统。

---

## 2. 目标

- 调酒主舞台更像真实桌面环境，而不是普通控件板。
- 量杯和成品杯保持清晰分工：左侧量杯承接 15ml 步进，右侧高球杯承接成品。
- 底部原料图像明显变大，按钮区缩小为辅助动作。
- iPhone 17 与 iPhone 17e 竖屏均能单屏完成所有调酒动作。

---

## 3. 范围

### 3.1 包含

- 替换调酒页空高球杯为写实玻璃杯试样。
- 调整 SpriteKit 中高球杯显示比例，避免与量杯重叠。
- 增加 SpriteKit 主舞台的台面线、环境光、杯底阴影。
- 补齐青柠片入杯渲染，强化杯中反馈。
- 增大底部原料行和原料图片占比。
- 缩小底部动作按钮高度、图标与文字字号。
- 按参考 UI 图重构为顶部胶囊 HUD、中央主舞台、底部玻璃原料托盘、单行动作栏。
- 使用 Apple Liquid Glass 方向的半透明材质、亮边、暗层和可读性保护。

### 3.2 不包含

- 新酒品。
- 新评分维度。
- 音效系统。
- 真实流体物理模拟。
- 正式美术资产管线。

---

## 4. 验收标准

- 调酒区中量杯和成品杯不互相压盖，视觉关系明确。
- 空高球杯比旧版更接近真实玻璃杯，杯口和厚底可辨认。
- 原料区贴近屏幕底部，原料图片比 P0.8 更大。
- 两行动作按钮仍可完整显示，且不会抢占原料视觉主次。
- 点击量杯型原料后，量杯内出现液面反馈。
- 点击“量杯入杯”后，底部已加数值更新，成品杯出现液体反馈。
- iPhone 17 与 iPhone 17e 模拟器截图中全部核心操作仍在单屏内。
- 底部左右两端原料不得溢出屏幕。
- 单行动作栏中 5 个核心动作均可见：量杯入杯、加冰、摇酒、出杯、清空量杯。
