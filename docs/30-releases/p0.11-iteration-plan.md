# mixcrit P0.11 迭代计划

> 制定日期：2026-06-18  
> 基线版本：`0.0.12 (12+)` — P0.10 首次体验收口

---

## 1. 迭代定位

P0.11 是 **调酒主玩法定稿版**。

它不是养成系统版本，而是进入养成系统之前的最后一轮核心玩法打磨：让 Mojito 调酒页在视觉、反馈、构图和小屏适配上足够稳定，后续 P1 才能把顾客、金币、经验、解锁等养成层建立在可靠的调酒体验上。

| 项目 | 值 |
|------|----|
| 产品阶段 | P0.11 |
| 发布标识 | 0.0.13 (13+) |
| 主题 | 写实调酒主舞台定稿 + 动作反馈强化 |
| Spec | `docs/20-specs/p0.11/realistic-mixing-stage-finalization.md` |
| 不进本版 | 经营系统、第二杯酒、顾客系统、金币经验 |

---

## 2. 为什么不是直接 P1.0

mixcrit 未来要做成养成类游戏，但 P1 的养成闭环依赖一个可复用的“调一杯酒”核心。

如果当前 Mojito 调酒页仍存在主舞台空洞、动作反馈弱、玩家不确定点击后发生了什么，那么提前加入经营系统会带来三个风险：

- 养成奖励无法掩盖核心操作不爽的问题。
- 后续每新增一杯酒都会重复修 UI 和动画债务。
- 经济循环参数缺少稳定基线，例如一杯酒耗时、失败点、评分体验。

因此 P0.11 先把调酒体验定稿，P0.12 再处理配方扩展，P1.0 才进入养成 MVP。
