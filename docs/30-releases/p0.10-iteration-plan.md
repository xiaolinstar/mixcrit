## 1. 迭代定位

P0.10 是 **P0 首次体验收口**，不扩展酒单与经营系统。

| 项目 | 值 |
|------|----|
| 产品阶段 | P0.10 |
| 发布标识 | 0.0.12 (12+) |
| 主题 | 新手引导 + 动态步骤提示 + 量杯流程可读性 |
| Spec | `docs/20-specs/p0.10/first-session-onboarding.md` |
| 不进本版 | P1 经营、第二杯酒、完整音效系统 |

**进入 P1 的前置条件**（本迭代结束时评估）：

- P0 验收清单全绿
- 至少 1 名非开发者无讲解完成 Mojito
- 团队判断「调一杯酒」好玩且可理解
